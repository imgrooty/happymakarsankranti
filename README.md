# Happy Makar Sankranti 🌞🪁

## Project Overview
A festive web application celebrating Makar Sankranti, featuring an interactive and animated experience that captures the spirit of this vibrant Indian harvest festival.

## 🎉 Features
- Responsive and animated hero section
- Floating til-gul (sesame-jaggery sweets) animations
- Flying kites with dynamic trails
- Decorative star effects
- Smooth scrolling interactions

## 🚀 Technologies Used
- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## 🛠️ Prerequisites
- Node.js (v18 or later)
- npm or yarn

## 🔧 Installation

1. Clone the repository
```bash
git clone https://github.com/your-username/happymakarsankranti.git
cd happymakarsankranti
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the development server
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 🌈 Customization
- Modify animation parameters in `components/hero-section.tsx`
- Adjust styles in Tailwind CSS configuration
- Add more interactive elements as needed

## 📱 Responsive Design
The application is fully responsive and works seamlessly across different device sizes.
